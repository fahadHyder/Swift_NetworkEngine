# Swift_NetworkEngine
NetworkEngine will help you to make API call easily with out the overhead of writing tons of boilerplate codes.
# Quick Start

In your [Podfile]:

```ruby
use_frameworks!

target "Change Me!" do
  pod "NetworkMediator"
end
```

NetworkEngine support Xcode 8.3, 9.0, 9.1, 9.2 and 9.3; Swift 3.1,
3.2, 3.3, 4.0 and 4.1 ;
